AvrCrafts is a modern, responsive e-commerce platform built with React and Firebase, featuring a clean user interface, secure authentication, and a comprehensive admin dashboard.

## Features

### User Features
- **Product Browsing**: Browse products with filtering and search functionality
- **Shopping Cart**: Add/remove items and manage quantities
- **User Authentication**: Secure signup and login with email/password
- **Wishlist**: Save favorite items for later
- **Responsive Design**: Works on all device sizes
- **Interactive UI**: Smooth animations and transitions with Framer Motion
- **Chatbot Support**: Built-in customer support chatbot

### Admin Dashboard
- **Dashboard**: Overview of store statistics
- **Product Management**: Add, edit, and remove products
- **Category Management**: Organize products into categories
- **Order Management**: View and process customer orders
- **Blog Management**: Create and manage blog content
- **Analytics**: View sales and user engagement metrics

## Technologies Used

- **Frontend**: 
  - React 18
  - React Router 6
  - Tailwind CSS
  - Framer Motion
  - Lucide Icons

- **Backend**:
  - Firebase Authentication
  - Firebase Firestore
  - Firebase Storage

- **Build Tool**: Vite
